<p>FFOS demo application</p>

<h1>How to run:</h1>
<ol>
<li>Git clone</li>
<li>Open project with IntelliJ</li>
<li>Run</li>
</ol>
<h1>Database:</h1>
<ul>If in memory:
    <li>Open http://localhost:8080/h2-console/</li>
    <li>JDBC URL: jdbc:h2:mem:ffos_db</li>
    <li>Username: sa</li>
    <li>Empty password</li>
    </ul>  
<ul>If file:
    <li>Open http://localhost:8080/h2-console/</li>
    <li>JDBC URL: jdbc:h2:file:./ffos</li>
    <li>Username: sa</li>
    <li>Empty password</li>
</ul>
