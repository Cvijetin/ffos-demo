# ffos-demo

How to run:
  1. Git clone
  2. Open project with IntelliJ
  3. Run
  
Database:
  If in memory:
    - Open http://localhost:8080/h2-console/
    - JDBC URL: jdbc:h2:mem:ffos_db
    - Username: sa
    - Empty password
  If file:
    - Open http://localhost:8080/h2-console/
    - JDBC URL: jdbc:h2:file:./ffos
    - Username: sa
    - Empty password
